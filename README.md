# Zomato Analytics Project

## Project Overview

This project analyzes Zomato restaurant data using Excel, Power BI, Tableau, and SQL to identify restaurant trends, customer preferences, pricing patterns, cuisine popularity, online delivery adoption, and table booking insights.

## Data Preparation

### Country Map Table

Created a Country Map table using Country Code and Country Name to enable country-wise analysis and reporting.

### Calendar Table

Built a Calendar Table using the DateKey column and derived the following attributes:

* Year
* Month Number
* Month Full Name
* Quarter (Q1, Q2, Q3, Q4)
* Year-Month (YYYY-MMM)
* Weekday Number
* Weekday Name
* Financial Month (April = FM1 to March = FM12)
* Financial Quarter
